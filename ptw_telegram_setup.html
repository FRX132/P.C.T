Telegram Signal-Einrichtung

Schritt-für-Schritt-Anleitung zur Signal-Einrichtung
Folge dieser einfachen Anleitung, um deinen eigenen Telegram-Bot zu erstellen und Signale direkt aus dem Terminal zu empfangen.

1 Telegram Bot erstellen (BotFather)

Der BotFather ist der offizielle Telegram-Bot zur Erstellung anderer Bots.

  • Suche in Telegram nach dem Benutzer @BotFather.
  • Starte den Chat und sende den Befehl /newbot.
  • Gib einen Namen für deinen Bot ein (z.B. PTW Signal Bot).
  • Wähle einen eindeutigen Username, der auf "bot" endet (z.B. ptw_signal_bot).
  • Kopiere das generierte HTTP API Token (z.B. 123456789:ABCdefGhI...). Bewahre dieses geheim auf!

2 Chat ID / Gruppen-ID herausfinden

Du musst herausfinden, wohin der Bot die Signale senden soll. Das kann dein privater Chat, eine Gruppe oder ein Kanal sein.

Option A: Privater Chat (Direktnachrichten)

  • Suche in Telegram nach deinem neu erstellten Bot und starte den Chat mit /start.
  • Suche nach dem Bot @userinfobot und starte ihn.
  • Er sendet dir sofort deine persönliche Chat ID (eine positive Zahl, z.B. 987654321).

Option B: Telegram Gruppe oder Kanal

  • Erstelle eine neue Gruppe oder einen Kanal in Telegram.
  • Füge deinen erstellten Bot als Mitglied / Administrator hinzu.
  • Sende eine beliebige Testnachricht in die Gruppe.
  • Rufe im Browser folgende URL auf (ersetze TOKEN durch deinen Bot-Token):
    https://api.telegram.org/botTOKEN/getUpdates
  • Suche im zurückgegebenen Text nach dem Abschnitt "chat":{"id": -100xxxxxxxxxx, ...}.
  • Die ID mit dem Minus-Zeichen (z.B. -100123456789) ist deine Gruppen- oder Kanal-ID.

3 Im Terminal eintragen & aktivieren

Sobald du die Daten hast, kannst du sie im Terminal speichern.

  • Gehe auf die Seite Einstellungen.
  • Trage deinen Telegram Bot Token und deine Chat ID in das Telegram-Panel ein.
  • Klicke auf Speichern.
  • Klicke auf Test-Signal senden, um die Verbindung sofort zu testen. Du solltest eine Nachricht in Telegram erhalten!
  • Aktiviere optional Auto-Senden bei Breakout, damit Breakout-Signale bei Klick auf BUY/SELL automatisch versendet werden.
